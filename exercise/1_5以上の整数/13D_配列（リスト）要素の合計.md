# 配列（リスト）要素の合計 (paizaランク D 相当)
## 下記の問題をプログラミングしてみよう！
**指定した配列（リスト）を定義し、配列（リスト）の要素すべてを全て加算して出力してください。**
- (\*編注: 指定された内容のリストを作成せよ、ということ)
  - つまり、書いてある通りの値でリストをハードコーディング（ベダ書きともいう）する

### 入力される値
```
なし
```

`1, 3, 5, 6, 3, 2, 5, 23, 2`  
を要素に持つ配列（リスト）をプログラムで定義し、使用すること。


入力値最終行の末尾に改行が１つ入ります。  
文字列は標準入力から渡されます。

### 期待する出力
```
配列（リスト）の要素全てを足した値。
```
最後は改行し、余計な文字、空行を含んではいけません。

- なし
